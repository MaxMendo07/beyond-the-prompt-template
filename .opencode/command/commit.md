descripcion: Añade los cambios del repositorio al staging area, haz commit con un mensaje claro y descriptivo que siga 'fase N descripcion'.
___

Añade los cambios al staging area:

```bash
git add -A
```

Haz el commit con un mensaje que siga formato `fase N descripcion`, donde:
- `N`es el numero de fase.
- `descripcion` es el resumen breve

```bash
git commit -m "fase N descripcion"
